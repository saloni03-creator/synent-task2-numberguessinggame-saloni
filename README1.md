# synent-task2-numberguessinggame-saloni
## Project Description

The Number Guessing Game is a simple Python command-line interface (CLI) project where the computer generates a random number, and the user tries to guess it. 
The program provides hints such as "Too High" or "Too Low" until the correct number is guessed.
This project is designed for beginners to practice Python basics like loops, conditional statements, user input, and the random module.


## Features

- Random number generation
- User input for guessing numbers
- Hint system ("Too High" / "Too Low")
- Unlimited attempts until correct guess
- Simple and interactive CLI gameplay


## Technologies Used

- Python 3
- Command Line Interface (CLI)


## How to Run

### Step 1:
Install Python 3 on your system.

### Step 2:
Save the Python file as:
number_guessing_game.py

### Step 3:
Open terminal or PyCharm and run:
python number_guessing_game.py
